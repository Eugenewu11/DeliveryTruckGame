# Delivery Truck Game

Un juego 2D de entregas desarrollado en Unity donde controlas un camión repartidor que debe recoger y entregar paquetes a los clientes.

## Descripción

Delivery Truck Game es un juego casual en el que manejas un vehículo de reparto por la ciudad. Tu objetivo es recoger helados y entregarlos a los clientes por todo el mapa. El juego incluye mecánicas de conducción arcade con controles simples y efectos visuales de partículas cada vez que recoges un paquete(helado).

## Mecánicas del Juego

### Controles
- **W**: Avanzar hacia adelante
- **S**: Retroceder
- **A**: Girar a la izquierda
- **D**: Girar a la derecha
- **W + A/D**: Avanzar mientras giras
- **S + A/D**: Retroceder mientras giras
- **Shift**: Acelerar (boost de velocidad)

### Sistema de Entregas
- Recoge los paquetes de helado en el mapa
- Al recoger un paquete, la velocidad del vehículo aumenta
- Entrega los paquetes a los clientes para completar la entrega
- Sistema de partículas que indica cuando llevas un paquete

### Características
- Controles de conducción arcade responsivos
- Sistema de velocidad variable con boost
- Efectos visuales con sistema de partículas
- Detección de colisiones para recoger y entregar paquetes

## Requisitos del Sistema

- Unity 6000.0.62f1 o superior
- Sistema operativo: Windows, macOS o Linux
- Espacio en disco: ~500 MB

## Instalación

### Para Desarrolladores (Proyecto Unity)
1. Clona este repositorio:
   ```bash
   git clone https://github.com/Eugenewu11/DeliveryTruckGame.git
   ```

2. Abre Unity Hub

3. Haz clic en "Add" o "Agregar" y selecciona la carpeta del proyecto

4. Asegúrate de tener instalada la versión de Unity 6000.0.62f1 o compatible

5. Abre el proyecto en Unity

6. Ve a la carpeta `Assets/Scenes` y abre la escena `SampleScene.unity`

7. Presiona el botón Play en el editor de Unity para probar el juego
